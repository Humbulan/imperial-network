Getting Started

Integrate Imperial Network into your mobile app using our SDK.

npm install imperial-mobile-sdk
Initialize SDK:
import ImperialSDK from 'imperial-mobile-sdk';

const imperial = new ImperialSDK({
    apiKey: 'YOUR_API_KEY',
    environment: 'production' // or 'sandbox'
});
